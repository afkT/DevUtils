
## Gradle

```gradle
// DevWidget - 自定义 View UI 库
implementation 'io.github.afkt:DevWidgetX:1.2.6'
```

## 目录结构

```
- dev                  | 根目录
   - widget            | 自定义 View 根目录
      - adjust         | 自动调节高度 View
      - assist         | View 辅助类
      - custom         | 自定义 View
      - decoration     | RecyclerView ItemDecoration
         - grid        | Grid ItemDecoration
         - linear      | Linear ItemDecoration
      - function       | 需求功能 View
      - ui             | UI View
         - radius      | 圆角相关 View
         - resize      | 自适应高度 ImageView
         - round       | 圆角相关 View
      - utils          | 工具类目录
```
